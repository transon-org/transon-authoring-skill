# Hygiene

Follow the tool-neutral body in `harness/skills/hygiene.md`.

Comments explain non-obvious intent only. Contract docs (`docs/SPEC.md`) carry **current state** —
no stacked `*(rev …)*` markers, no superseded designs. Traceability Tests cells = test references
only. Session narrative goes to `docs/current-state.md`. When editing, **delete** violating
comments/prose; never add a comment that narrates the change you are making.
